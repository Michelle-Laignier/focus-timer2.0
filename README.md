# Focus Timer 2.0
Temporizador estilo pomodoro feito pro curso Explorer da Rocketseat. <br>
Projeto feito pra praticar modularização, sons e alterar entre modo escuro e modo claro pelo JavaScript.

## Tecnologias
- HTML <br>
- CSS <br>
- JS <br>
- Git
